# PWM Examples

- `servo_basic`: host-runnable introduction to angle-based servo control
- `esc_basic`: host-runnable introduction to throttle-based ESC control
- `multi_servo_airframe`: multiple-servo control surface workflow for an aircraft
- `rp235x_servo_pico2`: RP2350 / Pico 2 style hardware example using `rp235x-hal`
- `rp235x_four_servo_pico2`: RP2350 example for four airframe servos across two PWM slices

If you are starting from scratch, read `servo_basic` first, then
`multi_servo_airframe`, then `rp235x_servo_pico2`, and finally the four-servo
RP2350 example once you are ready to wire a real aircraft-style setup.
